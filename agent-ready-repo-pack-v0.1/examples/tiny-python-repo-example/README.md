# Tiny Python Repo Example

A tiny example structure for testing the Agent-Ready Repo Pack.

```text
tiny-python-repo/
  AGENTS.md
  src/
    calculator.py
  tests/
    test_calculator.py
  README.md
```

## Example Task

```text
Add a subtract function to src/calculator.py and add one test.
```

## Expected Agent Behavior

- read `AGENTS.md`
- inspect `src/calculator.py`
- inspect tests
- add one small function
- add one small test
- run tests
- report changed files and verification
